The Tactile Highlight component emphasizes important inline text with a premium, spring-driven reveal. It supports directional highlights, viewport and hover triggers, animation delays, and a wide set of color variants for light and dark themes.

## Usage

### Installation

<InstallBlock command="tactile-highlight" />

### Basic Example

```tsx
import { TactileHighlight } from "@/components/tactile-highlight";

export default function App() {
  return (
    <p>
      Our mission is to build{" "}
      <TactileHighlight direction="left" trigger="inView" color="default">
        Better Interfaces.
      </TactileHighlight>
    </p>
  );
}
```

### Trigger Modes

Use `trigger="inView"` to animate when the text enters the viewport and reset when it leaves. Use `trigger="auto"` to reveal immediately, or `trigger="hover"` to reveal on hover.
